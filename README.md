# 《广韵》反切系联工具 使用说明

本工具基于陈澧《切韵考》的反切系联法，对《广韵》所收反切进行系联分析，以可视化的方式呈现**上字系联（声类）**与**下字系联（韵类）**的关系。所有数据离线运行，无需联网，无需安装。

> 数据底本：《广韵》反切。系联算法遵循陈澧"同用、互用、递用"三条原则。

---

## 一、如何打开

### 在线访问（推荐）
访问部署后的网址即可，例如：
```
https://你的用户名.github.io/仓库名/
```

### 本地打开
1. 下载本仓库的两个文件到**同一文件夹**：
   - `index.html`（主程序，约 4.2 MB）
   - `系联流程图_data.js`（流程图数据，约 3.8 MB）
2. 用现代浏览器（Chrome / Edge / Firefox）双击打开 `index.html` 即可。
   > 注意：必须保证两个文件在同一目录，否则"系联流程图"功能将无法显示。

## 二、五个功能模块

页面顶部有 5 个标签页（Tab），点击即可切换：

### 1. 反切查询
- **用途**：查任一汉字在《广韵》中的反切及音韵地位。
- **操作**：在搜索框输入汉字（支持繁简字），点"查询"或回车。
- **结果展示**：
  - 该字所有反切条目（反切上字 + 下字 + 韵目 + 声组 + 开合 + 等第）。
  - 可展开"同声类上字"卡片，查看该字所属声类下的全部上字。

### 2. 总览
- **用途**：一览全部声类与韵目的整体分布。
- **操作**：
  - 默认折叠，点击标题栏（▼/▶）即可展开。
  - **声类一览**：陈澧所分四十声类及其所含上字。
  - **韵目一览**：《广韵》206 韵及各组韵目。

### 3. 上字系联
- **用途**：查看反切上字如何系联成声类。
- **操作**：
  - 在搜索框输入上字（如"都"），即时筛选。
  - 点击左侧"声类列表"中任一声类（如"端类"），查看：
    - 该声类下所有上字。
    - **系联关系**：同用 / 互用 / 递用 三种关系列表。
    - 节点关系图（Canvas 绘制，圆形布局，可拖拽）。

### 4. 下字系联
- **用途**：查看反切下字如何系联成韵类。
- **操作**：
  - 点击"选择韵目"展开，从韵目网格中点选任一韵（如"东"）。
  - 搜索框可输入小韵首字或下字快速定位。
  - 查看该韵下：
    - 全部下字及所属小韵。
    - 系联关系（同用/互用/递用）。
    - 节点关系图。

### 5. 系联流程图
- **用途**：以交互式流程图总览上字、下字、韵目之间的关联。
- **操作**：
  - 左侧栏分两组：
    - **上字（按发音部位）**：唇/舌/齿/牙/喉/半舌半齿。
    - **下字（按韵摄）**：通/江/止/遇/蟹/效/果/假/宕/梗/曾/流/深/咸。
  - 点击任一节点，右侧 Canvas 即绘制该节点的系联图。
  - 搜索框可输入"上字 / 下字 / 韵目 / 声组"任意关键词筛选。
- **图形交互**：
  - 鼠标拖拽节点可移动位置。
  - 鼠标悬停节点显示详情。
  - 节点圆形布局，便于看清关联结构。

## 三、系联关系说明

陈澧《切韵考》反切系联三例：

| 关系 | 含义 | 例子 |
|---|---|---|
| **同用** | 二字同被第三字用作反切上字（或下字） | 都、当 同为丁的反切上字 |
| **互用** | 二字互为反切上字（或下字） | 都 当 / 当 都 |
| **递用** | 甲用乙为反切上字，乙又用丙 | 甲→乙→丙 |

工具在"上字系联""下字系联"中以列表形式给出这三种关系，并在右侧 Canvas 以节点连线图直观呈现。

## 四、常见问题

| 问题 | 解决 |
|---|---|
| 打开"系联流程图"为空白 | 检查 `系联流程图_data.js` 是否与 `index.html` 在同一目录。 |
| 中文显示为乱码 | 浏览器编码设置应为 UTF-8（现代浏览器默认）。 |
| 图形无法拖拽 | 确保使用鼠标左键按住节点拖动。 |
| 手机访问字太小 | 浏览器可双指缩放；推荐电脑访问以获最佳体验。 |
| 国内访问 GitHub Pages 慢 | 可改用 Gitee Pages 镜像，或下载到本地打开。 |
